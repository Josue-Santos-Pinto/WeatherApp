# Iniciando projeto

_npm install_ <br>
_npm start_

## Sobre o aplicativo

Aplicativo criado para consulta de informações climaticas sobre a cidade desejada

## Tecnologias utilizadas

_React-Native_,
_Expo_,
_Typescript_,
_React-Query_,
_Axios_,
_RestAPI_,
_UnsplashAPI_,
_OpenWeatherAPI_
